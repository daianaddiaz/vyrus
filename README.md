# 🧫 VYRUS

**VYRUS** es un videojuego de simulación y estrategia biológica desarrollado para la materia **Programación de Videojuegos I** de la **Universidad Nacional de Hurlingham (UNAHUR)**.  
El proyecto está siendo realizado por **Daiana Diaz y Pablo Coronel**.

---

## 🎮 Sobre el juego

En el fondo de una placa de Petri, una colonia de bacterias lucha desesperadamente por su supervivencia. El jugador asume el rol de un biólogo de laboratorio en una carrera contra el tiempo: deberá contener una agresiva y mutante infección viral que amenaza con consumir toda la muestra. 

A través del despliegue táctico de **torres de sanidad** y la contención de los peligrosos **Virus Alfa**, el objetivo es mantener con vida a la población celular durante 60 agónicos segundos. ¡Cada bacteria cuenta para salvar la simulación!

## 🛠️ Tecnologías utilizadas

* **JavaScript (ES6+)** - Lógica modular, sistemas de colisiones y motor físico
* **PIXI.js (v8)** - Renderizado gráfico de alto rendimiento a 60 FPS mediante WebGL
* **Adobe Illustrator/ Canva / Piskel** - Diseño de interfaz pixel-art, patrones, y hojas de sprites animados.
* **LocalStorage** - Sistema de guardado local de datos para el ranking
* **Fuente** - Google Fonts.
